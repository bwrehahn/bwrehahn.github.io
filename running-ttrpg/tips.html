Tips for Success

This area is just for advice to playing your game. Some of this is repeated from previous sections but it can be really helpful, but this is just some advice you can take or leave.

  • Work together - The game is a collaborative effort on both the Game Master and the Players. The Game Master and Players can exchange ideas and the Game Master can incorporate them into the game, and the Game Master can create "House Rules" to make the game more enjoyable.
  • Don't think of it as a game -As mentioned previously, this is a game that tells a story, so think of it as a collabrative, interactive story. If you start thinking of it as strictly a game, someone will try to win and try to interfere with the other players and that could cause the group to not have fun.

  • Have fun! This has been mentioned several times before so by now it is self-explanatory.

  • House Rules - Do not be afraid to implement them. As long as you and all the players are aware of them, there is nothing wrong with them. No one from the game company is going to come along and say you cannot do that.

  • Game Book Resources - Many of the game books come with some ideas and monsters and quests you can use, so do not be afraid to use these ideas. Do not feel you have to create everything yourself.

  • Do not push - Some Game Masters have a tendency to push the Players to do some things the Players do not want to do. You need to learn to guide the Players to make them want to do it. It will take some practice to do so comfortably.

  • Extras - This was mentioned before but having some extra supplies on hand can be a great thing.

If you have gone through all the other pages, thank you and I hope this has been very helpful in your pursuits of Game Mastering.
